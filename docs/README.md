# Documentation

Reference docs for **lotr-sdk**. Start with the project [README](../README.md) for installation,
quickstart, authentication, and configuration; this folder holds the detailed references.

- **[endpoints.md](endpoints.md)** — the movie and quote endpoints, with real request/response
  examples for each (`/movie`, `/movie/{id}`, `/movie/{id}/quote`, `/quote`, `/quote/{id}`).
- **[querying.md](querying.md)** — the `Query` builder: every filter operator, sorting, and
  pagination, with the readable and on-the-wire form of each.

For the architecture and design rationale, see [design.md](../design.md).
